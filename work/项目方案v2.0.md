石油行业巡检与设备管理系统 - 项目方案 v2.0

========================================
一、项目概述
========================================

1.1 项目背景
面向石油行业（采油、炼化、储运等场景），构建一套覆盖移动端巡检+设备检查+PC调度室大屏的综合性管理平台。

1.2 核心用户角色
- 超级管理员：系统全局配置、用户管理、角色权限分配
- 初级管理员：巡检计划配置、巡检点管理、巡检监督
- 巡检员：日常巡检执行、NFC打卡、隐患上报
- 设备检查员/维保人员：设备扫码检查、维护保养操作
- 调度室值班员：大屏监控、当日状态查看、异常处理跟踪
- 审核员：隐患上报审核、处理验收

========================================
二、系统架构总览
========================================

PC调度室端(Web - Apache托管)
    | HTTPS
Apache HTTP Server (反向代理 + SSL)
    | 反向代理
FastAPI后端服务 (:8000内部端口)
    |
MySQL 8.x + Redis + 文件存储
    | HTTPS
Android移动端App
 - NFC巡检模块
 - 设备扫码模块
 - 隐患上报模块(语音转文字+视频)
 - 离线缓存

========================================
三、功能模块详细设计
========================================

--- 模块一：移动端NFC巡检系统 ---

1) 巡检点管理（初级管理员/PC端配置）
   - 名称、编号、NFC标签UID、GPS坐标、所属区域
   - 关联岗位：该巡检点由哪些岗位负责巡检
   - 状态：启用/停用/维护中
   - 批量Excel导入 + NFC标签批量写入

2) 巡检岗位管理
   - 岗位名称、班次类型（白班/中班/夜班/24h）
   - 关联人员：一个岗位可有多个巡检员
   - 关联巡检点：该岗位负责哪些巡检点
   - 多岗位管理，支持上下级关系（总站->分站->班组）

3) 巡检规则配置
   - 检查时间窗：默认+-30分钟
   - 超时判定：超出时间窗60分钟标记超时
   - 提前检查限制：早于时间窗30分钟（防补签）
   - 漏检判定：当日23:59:59仍未检查标记漏检
   - 巡检周期：每日/每周/每月/自定义
   - 一键生成月度计划

4) 巡检员操作界面（移动端）
   - 今日巡检进度：已检/未检/超时/提前，颜色标识
   - NFC打卡：靠近巡检点NFC标签自动识别
   - 手动输入NFC ID备用
   - 拍照记录（时间水印+GPS水印）
   - 隐患上报：拍照+视频+语音转文字

5) 隐患上报闭环流程
   第1步: 上报（巡检员）-> 状态:待审核
     拍照/录像/语音转文字/选择类型和紧急程度
   第2步: 审核（审核员）-> 已确认/已驳回
   第3步: 处理（维保人员）-> 待验收
     处理措施描述+处理后拍照
   第4步: 验收（审核员）-> 已关闭
     通过验收或退回重处理

6) 离线支持
   - 网络不佳时本地缓存，恢复后自动同步
   - 离线期间NFC功能正常工作

--- 模块二：移动端设备检查模块 ---

1) 设备二维码管理
   - PC端为每台设备生成唯一二维码
   - 支持批量打印二维码标签
   - 设备档案：编号、名称、型号、厂家、安装日期、位置、运行参数

2) 设备扫码检查流程
   扫描设备二维码 -> 获取设备信息和检查任务 -> 逐项检查 -> 异常自动转隐患上报 -> 提交记录

3) 检查项类型
   - 单选：正常/异常/需清洁
   - 数值：输入具体数值，设定正常范围
   - 文本：自由描述
   - 拍照：必须上传照片

--- 模块三：PC端调度室实时监控大屏 ---

1) 大屏布局（深色科技风）
   - 站点平面图：标注巡检点和设备位置
   - 巡检进度：今日完成率、已检/未检/超时/提前
   - 岗位完成情况：各岗位进度条排名
   - 设备状态：正常运行/异常/维修中/停机
   - 实时告警：最新隐患滚动列表，红黄橙三级
   - 今日统计：完成率、隐患上报数、关闭数

2) 巡检监控页
   - 当日概览：各岗位是否全部巡检完成
   - 未完成明细：精确到岗位、员工、巡检点
   - 异常巡检内容列表（带照片缩略图）

--- 模块四：PC端后台管理系统 ---

1) 仪表盘（首页）
   - 用户总数、巡检点总数、设备总数、今日隐患
   - 今日待办：待审核隐患、待处理任务、超时预警

2) 巡检管理
   - 巡检点管理：CRUD + 批量导入 + NFC绑定
   - 岗位管理：CRUD + 人员分配 + 班次设置
   - 巡检计划管理：月度计划日历视图
   - 巡检监督：实时进度 + 漏检/超时/提前预警

3) 设备管理
   - 设备档案CRUD + 批量Excel导入
   - 二维码生成与打印
   - 设备参数配置
   - 检查项配置（可自定义添加）
   - 维保管理：保养周期 + 保养提醒 + 历史记录

4) 检查/维保任务管理
   - 创建检查任务：手动添加，指定岗位/人员
   - 检查内容：自由添加检查项
   - 频率设置：每日/每周/每月/自定义
   - Excel批量导入检查任务
   - 任务执行跟踪：当日完成情况 + 未完成明细

5) 隐患管理
   - 隐患列表 + 状态流转（待审核->已确认->处理中->待验收->已关闭）
   - 隐患统计：按类型/紧急程度/区域分布
   - 趋势分析 + 平均处理时长

6) 统计分析
   - 巡检统计：日/周/月完成率、效率排名
   - 设备统计：完好率、故障率、维保及时率
   - 隐患统计：总数/关闭率、类型占比、趋势
   - 报表导出：Excel/PDF

7) 用户与权限管理（RBAC）
   - 用户CRUD + 部门树形结构
   - 角色：超级管理员/初级管理员/巡检员/调度员/维保人员/审核员
   - 按角色分配菜单和功能权限

8) 系统配置
   - 巡检规则：超时/提前/漏检阈值
   - 通知配置：超时提醒/告警推送
   - 数据配置：照片/视频保留期限、备份策略
