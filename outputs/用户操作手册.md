# 石油行业巡检与设备管理系统 — 用户操作手册

## 一、系统登录

1. 浏览器访问系统地址（如 `http://服务器IP`）
2. 输入用户名和密码
3. 默认管理员账号：`admin` / `admin123`（首次登录务必修改密码）

## 二、角色与权限

| 角色 | 可见范围 | 主要功能 |
|------|---------|---------|
| 公司管理员 | 全部 | 所有功能，跨大队管理 |
| 大队管理员 | 本大队 | 本大队的人员、巡检、设备管理 |
| 班组长 | 本班组 | 班组成员管理、巡检监督 |
| 巡检员 | 个人 | 执行巡检、隐患上报（移动端） |
| 设备维保员 | 分配任务 | 设备检查、隐患处理 |
| 审核员 | 全部（只读） | 隐患审核验收、大屏查看 |
| 调度员 | 全部（只读） | 大屏查看、报表查看 |

## 三、系统管理

### 部门管理
**路径**: 系统管理 → 部门管理

- 部门层级：公司 → 大队 → 班组
- 点击"新增"添加部门，选择上级部门
- 支持树形展开/折叠

### 用户管理
**路径**: 系统管理 → 用户管理

- 创建用户：填写用户名、姓名、工号、手机、选择所属部门和角色
- 用户角色决定其可见范围

### 角色权限
**路径**: 系统管理 → 角色权限

- 预置角色：公司管理员、大队管理员、班组长、巡检员、设备维保员、审核员、调度员
- 可按需自定义角色

## 四、巡检管理

### 巡检点管理
**路径**: 巡检管理 → 巡检点管理

1. 新增巡检点：填写名称、编号、选择所属部门、GPS坐标
2. NFC绑定：为巡检点配置NFC标签UID
3. 批量导入：下载模板 → 填写 → 上传Excel
4. 巡检点启用/停用控制

### 岗位管理
**路径**: 巡检管理 → 岗位管理

- 创建岗位：名称、班次（白班/中班/夜班/24h）
- 关联巡检点和人员

### 巡检计划
**路径**: 巡检管理 → 巡检计划

- 系统每月1日自动生成月度计划
- 日历视图查看每日计划
- 支持手动调整

### 巡检监督
**路径**: 巡检管理 → 巡检监督

- 实时查看各班组巡检进度
- 未完成/超时/漏检明细
- 异常记录（含照片预览）

## 五、设备管理

### 设备档案
**路径**: 设备管理 → 设备档案

1. 新增设备：编号、名称、型号、厂家、部门、位置、类别
2. 检查项管理：点击"检查项"为设备配置检查清单
   - **单选型**: 配置选项（正常/异常等）
   - **数值型**: 配置正常范围（如温度 20-80℃）和单位
   - **文本型**: 自由输入
   - **拍照型**: 必须拍照上传
3. 二维码：点击"二维码"生成并打印设备标签
4. Excel批量导入

### 检查任务
**路径**: 设备管理 → 检查任务

1. 新建任务：名称、负责班组、任务类型（定期/一次性）、频率
2. 关联设备：多选关联设备
3. 执行跟踪：查看各班组完成情况

### 检查结果
**路径**: 设备管理 → 检查结果

- 按任务/设备筛选查看检查记录
- 正常/异常标记
- 详情弹窗含照片

## 六、隐患管理

### 隐患四步闭环流程

```
上报(巡检员) → 审核(审核员) → 处理(维保员) → 验收(审核员)
```

### 隐患列表
**路径**: 隐患管理 → 隐患列表

- 顶部统计卡片：总数、待审核、待处理、已完成、已关闭
- 支持按状态、紧急程度、关键词筛选
- 点击行查看详情：含照片/视频附件、流转时间线
- 操作按钮：
  - **审核** — 通过（进入处理中）/ 驳回（直接关闭）
  - **处理** — 填写处理结果并提交
  - **验收** — 确认处理结果，关闭隐患

### 隐患审核
**路径**: 隐患管理 → 隐患审核

- 集中展示所有待审核隐患
- 查看详细信息和附件后
- 一键通过或驳回（驳回需填写原因）

## 七、统计分析

### 巡检统计
**路径**: 统计分析 → 巡检统计

- 今日概要：应检/已检/完成率/超时漏检
- 各班组完成率柱状图（颜色区分）
- 近7天趋势（完成率折线 + 应检数柱状）
- 巡检员排名
- 支持导出Excel

### 设备统计
**路径**: 统计分析 → 设备统计

- 设备总数/正常运行/维修报废/完好率
- 状态分布饼图
- 各类别正常/异常堆叠柱状图
- 设备明细清单导出

### 隐患统计
**路径**: 统计分析 → 隐患统计

- 隐患总数/待审核/处理中/已关闭
- 紧急程度分布饼图
- 近12个月上报/关闭趋势
- 最近隐患记录导出

## 八、实时监控大屏

**路径**: 实时监控大屏

- 深色科技风全屏仪表盘
- 6组KPI指标卡
- 巡检进度图 + 班组排名
- 实时告警滚动列表
- 设备状态饼图 + 巡检点分布
- 今日巡检明细表
- 左上角切换大队/班组视图
- WebSocket 实时数据推送

## 九、移动端操作（Android App）

### NFC巡检
1. 登录后查看"今日待巡检"列表
2. 到达巡检点，手机靠近NFC标签
3. 自动识别巡检点，拍照记录
4. 正常提交或发现隐患上报

### 隐患上报
1. 选择隐患类型和紧急程度
2. 拍照（最多5张）+ 录像（最多3段，每段≤60s）
3. 填写文字描述
4. 提交 → 自动进入审核流程

### 设备扫码检查
1. 扫描设备二维码
2. 加载设备和检查任务
3. 逐项检查并记录结果
4. 异常自动转隐患

### 离线支持
- 网络不佳时自动缓存本地
- NFC功能离线可用
- 恢复网络后自动同步

## 十、常见问题

**Q: NFC标签如何绑定？**
A: PC端"巡检点管理"→ 编辑巡检点 → 填入NFC标签UID

**Q: 月度计划未生成？**
A: 系统每月1日凌晨2点自动生成。也可手动触发

**Q: 视频上传失败？**
A: 单个视频≤50MB，时长≤60秒，格式支持MP4/MOV/WebM

**Q: 如何修改巡检时间窗规则？**
A: 系统管理 → 角色权限 → 系统配置 → 巡检规则
